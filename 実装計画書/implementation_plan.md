# キャンフェス（文化祭）特設Webサイト 実装計画

## 概要

高校・大学の文化祭（キャンフェス）向けの特設Webサイトを HTML5 + CSS3 のみで構築します。
JavaScriptは一切使用せず、スマホ完全対応（レスポンシブ）の縦並びレイアウトをベースとします。
新メンバーが「コードの教科書」として使いやすい、読みやすい・編集しやすいコードを目指します。

---

## フォルダ構成

```
/（ワークスペースルート）
├── index.html                          ← メインページ
├── css/
│   └── common.css                      ← 共通スタイル（ナビゲーション・リセット等）
└── pages/
    └── sample-project/
        ├── index.html                  ← 企画紹介サンプルページ
        └── style.css                   ← 企画独自のスタイル
```

## 各ファイルの設計詳細

### [NEW] css/common.css

**役割**: サイト全体の土台・共通スタイル

- **フォント**: `Noto Sans JP`（Google Fonts）+ `sans-serif` フォールバック
- **リセット**: `box-sizing: border-box`、`margin: 0`、`padding: 0`
- **カラーパレット（CSS変数で定義）**:
  - `--color-primary`: `#FF6B6B`（珊瑚ピンク → 文化祭の活気）
  - `--color-secondary`: `#4ECDC4`（ティールグリーン）
  - `--color-accent`: `#FFE66D`（レモンイエロー）
  - `--color-dark`: `#2D3436`（テキスト）
  - `--color-light`: `#F9FAFB`（背景）
- **ナビゲーションバー**:
  - `position: fixed; top: 0`で画面最上部に固定
  - 高さ約56px、グラスモーフィズム風の半透明背景
  - リンク：「🏠 トップ」「📋 企画一覧」「📣 お知らせ」「⏰ タイムテーブル」
  - スマホでは横スクロール可能なフレキシブルナビ（JSなし）
- **共通ユーティリティクラス**:
  - `.container`（最大幅1100px、中央寄せ）
  - `.section`（上下パディング設定済みセクション）
  - `.btn`（共通ボタンスタイル）
  - `.card`（カード型コンテナ）

### [NEW] index.html（メインページ）

**読み込み**: `css/common.css`

**セクション構成（上から順）**:

1. **ナビゲーションバー** (`common.css`より)
2. **ヒーローセクション** 
   - グラデーション背景（primary → secondary）
   - 文化祭のキャッチコピー・日程・場所
   - 「企画一覧を見る」CTAボタン
   - CSSアニメーションで浮遊するデコレーション要素（⭐🎉🎊）
3. **お知らせ一覧セクション**
   - ダミー3件（例：「〇〇企画が追加されました」）
   - カード形式で表示
4. **タイムスケジュールセクション**
   - CSSのみのタイムライン表示（縦線 + 時刻 + イベント名）
   - ダミーデータ（開会式、午前の部、昼食休憩、午後の部、フィナーレ等）
5. **企画一覧へのリンクセクション**
   - サンプル企画カード（`pages/sample-project/index.html`へのデモリンク）
   - 「企画ページを追加するには？」という説明テキスト
6. **フッター**
   - 文化祭名・年度・クレジット

### [NEW] pages/sample-project/index.html（企画紹介サンプル）

**読み込み**: `../../css/common.css` + `./style.css`

**テーマ**: ☕ カフェ風おしゃれデザイン（「キャンパスカフェ」という架空の企画）

**セクション構成**:

1. **ナビゲーションバー** (common.css共通)
2. **企画ヒーローヘッダー**
   - 企画名「Campus Café ☕」
   - 開催場所（例：「3号館 2F 教室」）
   - 概要キャッチコピー
3. **企画概要・紹介文セクション**（150文字前後のダミーテキスト）
4. **メニュー（サービス内容）セクション**
   - カード形式で商品3種類（商品名・値段・説明）
   - 画像枠（`<img>` プレースホルダー）
   - 商品例：本日のコーヒー ¥200 / 手作りケーキ ¥300 / 季節のスムージー ¥250
5. **アレルギー情報セクション**
   - テーブル形式（商品名 × アレルゲン）
   - 卵・乳・小麦・ナッツ等
6. **タイムスケジュールセクション**
   - 営業時間・休憩時間のダミーデータ
7. **フッター**

### [NEW] pages/sample-project/style.css（企画独自スタイル）

**テーマ**: ナチュラルカフェ風

- **背景色**: クリーム `#FFF8F0`
- **文字色**: チョコレートブラウン `#3E2723`
- **アクセント**: ラテブラウン `#8D6E63` / ゴールド `#F9A825`
- **フォント**: `Playfair Display`（見出し）+ `Noto Sans JP`（本文）
- **ボタン**: 角丸・ブラウン系グラデーション・ホバー効果（CSSのみ）
- **カード**: 白背景・木目風ボーダー・影
- **セクション区切り**: ウェーブ形SVGボーダー（CSS clip-pathで実現）

## デザイン方針

| 要素 | 方針 |
|---|---|
| JavaScript | 一切使用しない |
| レスポンシブ | スマホ優先（max-width: 600px 基準） |
| レイアウト | 基本1カラム縦並び |
| アニメーション | CSS `@keyframes`のみ |
| 画像 | `<img>` + `alt` で画像枠として機能させる |
| コメント | 各セクションに日本語コメントで「ここを変えると○○が変わります」などの説明 |
